```java
public class XxxTurretBlock extends BaseEntityBlock {
    public static final DirectionProperty FACING = BlockStateProperties.FACING;
    
    // 1. 构造函数
    public XxxTurretBlock(Properties properties) {
        super(properties);
        this.registerDefaultState(this.stateDefinition.any().setValue(FACING, Direction.UP));
    }
    
    // 2. 放置检测 - 必须检测是否放在基座上
    @Override
    public @Nullable BlockState getStateForPlacement(BlockPlaceContext context) {
        Direction clickedFace = context.getClickedFace();
        BlockPos clickedPos = context.getClickedPos();
        BlockPos basePos = clickedPos.relative(clickedFace.getOpposite());
        Level level = context.getLevel();
        
        BlockEntity blockEntity = level.getBlockEntity(basePos);
        if (blockEntity instanceof TurretBaseBlockEntity) {
            return this.defaultBlockState().setValue(FACING, clickedFace);
        }
        return null; // 不是基座则不允许放置
    }
    
    // 3. 基座移除时炮塔也移除
    @Override
    public @NotNull BlockState updateShape(@NotNull BlockState state, @NotNull Direction direction, 
            @NotNull BlockState neighborState, @NotNull LevelAccessor level, 
            @NotNull BlockPos pos, @NotNull BlockPos neighborPos) {
        Direction facing = state.getValue(FACING);
        if (direction == facing.getOpposite()) {
            BlockPos basePos = pos.relative(facing.getOpposite());
            if (!(level.getBlockEntity(basePos) instanceof TurretBaseBlockEntity)) {
                return Blocks.AIR.defaultBlockState();
            }
        }
        return super.updateShape(state, direction, neighborState, level, pos, neighborPos);
    }
    
    // 4. 右键打开基座GUI
    @Override
    public @NotNull InteractionResult use(@NotNull BlockState state, @NotNull Level level, 
            @NotNull BlockPos pos, @NotNull Player player, @NotNull InteractionHand hand, 
            @NotNull BlockHitResult hit) {
        if (!level.isClientSide) {
            Direction facing = state.getValue(FACING);
            BlockPos basePos = pos.relative(facing.getOpposite());
            BlockEntity blockEntity = level.getBlockEntity(basePos);
            if (blockEntity instanceof TurretBaseBlockEntity baseEntity) {
                return baseEntity.getBlockState().getBlock().use(
                    baseEntity.getBlockState(), level, basePos, player, hand,
                    new BlockHitResult(hit.getLocation(), facing.getOpposite(), basePos, hit.isInside())
                );
            }
        }
        return InteractionResult.SUCCESS;
    }
    
    // 5. BlockEntity创建
    @Override
    public @Nullable BlockEntity newBlockEntity(@NotNull BlockPos pos, @NotNull BlockState state) {
        return new XxxTurretBlockEntity(pos, state);
    }
    
    // 6. tick方法注册
    @Override
    public @Nullable <T extends BlockEntity> BlockEntityTicker<T> getTicker(
            @NotNull Level level, @NotNull BlockState state, @NotNull BlockEntityType<T> type) {
        return level.isClientSide ? null : createTickerHelper(
            type, ModBlockEntities.XXX_TURRET.get(), XxxTurretBlockEntity::tick
        );
    }
    
    // 7. 渲染类型
    @Override
    public @NotNull RenderShape getRenderShape(@NotNull BlockState state) {
        return RenderShape.ENTITYBLOCK_ANIMATED;
    }
}
```

```java
public class XxxTurretBlockEntity extends BlockEntity implements GeoBlockEntity {
    
    // ========== 常量 ==========
    public static final int FIRE_RATE = 100;           // 射击间隔(tick)
    public static final double SEARCH_RADIUS = 48.0;   // 搜索范围
    public static final double BULLET_SPEED = 2.0;     // 子弹速度
    public static final float BULLET_DAMAGE = 10.0F;   // 子弹伤害
    public static final int ENERGY_COST = 5000;        // 能量消耗
    
    // ========== GeckoLib数据票（必须声明为static） ==========
    public static SerializableDataTicket<Boolean> HAS_TARGET;
    public static SerializableDataTicket<Double> TARGET_POS_X;
    public static SerializableDataTicket<Double> TARGET_POS_Y;
    public static SerializableDataTicket<Double> TARGET_POS_Z;
    
    // ========== 动画缓存 ==========
    private final AnimatableInstanceCache cache = GeckoLibUtil.createInstanceCache(this);
    
    // ========== 字段 ==========
    private int cooldown = 0;
    private LivingEntity target = null;
    public float yRot0 = 0.0f;  // yaw角度（GeoModel需要）
    public float xRot0 = 0.0f;  // pitch角度（GeoModel需要）
    
    // ========== 构造函数 ==========
    public XxxTurretBlockEntity(BlockPos pos, BlockState state) {
        super(ModBlockEntities.XXX_TURRET.get(), pos, state);
    }
    
    // ========== tick方法（服务端逻辑） ==========
    public static void tick(Level level, BlockPos pos, BlockState state, XxxTurretBlockEntity blockEntity) {
        if (level.isClientSide) return;
        
        TurretBaseBlockEntity base = blockEntity.getBaseEntity();
        if (base == null) return;
        
        // 无电量低头动画
        if (base.getEnergyStored() == 0) {
            blockEntity.target = null;
            blockEntity.setAnimData(HAS_TARGET, true);
            blockEntity.setAnimData(TARGET_POS_X, pos.getX() + 0.5);
            blockEntity.setAnimData(TARGET_POS_Y, pos.getY() - 2.0);
            blockEntity.setAnimData(TARGET_POS_Z, pos.getZ() + 0.5);
            return;
        }
        
        Direction facing = state.getValue(XxxTurretBlock.FACING);
        if (!base.isFaceEnabled(facing)) {
            blockEntity.target = null;
            blockEntity.setAnimData(HAS_TARGET, false);
            return;
        }
        
        if (blockEntity.cooldown > 0) {
            blockEntity.cooldown--;
        }
        
        blockEntity.updateTarget(level, pos, base, facing);
        
        if (blockEntity.target != null && blockEntity.cooldown <= 0) {
            if (blockEntity.canShoot(base)) {
                blockEntity.shoot(level, pos, state, base);
                blockEntity.cooldown = base.getFireRateForFace(facing, FIRE_RATE);
            }
        }
    }
    
    // ========== 获取基座 ==========
    public TurretBaseBlockEntity getBaseEntity() {
        Level level = getLevel();
        if (level == null) return null;
        BlockState state = getBlockState();
        if (!(state.getBlock() instanceof XxxTurretBlock)) return null;
        Direction facing = state.getValue(XxxTurretBlock.FACING);
        BlockPos basePos = worldPosition.relative(facing.getOpposite());
        BlockEntity blockEntity = level.getBlockEntity(basePos);
        if (blockEntity instanceof TurretBaseBlockEntity base) {
            return base;
        }
        return null;
    }
    
    // ========== 检查是否可以射击 ==========
    private boolean canShoot(TurretBaseBlockEntity base) {
        Direction facing = getBlockState().getValue(XxxTurretBlock.FACING);
        int energyCost = base.getEnergyCostForFace(facing, ENERGY_COST);
        
        // 检查能量
        if (base.getEnergyStored() < energyCost) return false;
        
        // 检查弹药（如果需要弹药）
        return hasAmmo(base);
    }
    
    // ========== 弹药检查（如果需要弹药） ==========
    private boolean hasAmmo(TurretBaseBlockEntity base) {
        IItemHandler ammoInv = base.getAmmoInventory();
        for (int i = 0; i < ammoInv.getSlots(); i++) {
            ItemStack stack = ammoInv.getStackInSlot(i);
            if (!stack.isEmpty() && stack.is(ModItems.XXX_AMMO.get())) {
                return true;
            }
        }
        return false;
    }
    
    // ========== 弹药消耗 ==========
    private void consumeAmmo(TurretBaseBlockEntity base) {
        IItemHandler ammoInv = base.getAmmoInventory();
        for (int i = 0; i < ammoInv.getSlots(); i++) {
            ItemStack stack = ammoInv.getStackInSlot(i);
            if (!stack.isEmpty() && stack.is(ModItems.XXX_AMMO.get())) {
                stack.shrink(1);
                return;
            }
        }
    }
    
    // ========== 执行射击 ==========
    private void shoot(Level level, BlockPos pos, BlockState state, TurretBaseBlockEntity base) {
        Direction facing = state.getValue(XxxTurretBlock.FACING);
        int energyCost = base.getEnergyCostForFace(facing, ENERGY_COST);
        if (base.getEnergyStored() < energyCost) return;
        if (!hasAmmo(base)) return;  // 再次检查弹药
        
        Vec3 muzzlePos = calculateMuzzlePosition(pos, facing);
        Vec3 targetPos = target.position().add(0, target.getEyeHeight() * 0.5, 0);
        
        // 预判瞄准
        if (base.isPredictiveAiming()) {
            double dist = muzzlePos.distanceTo(targetPos);
            double time = dist / BULLET_SPEED;
            targetPos = targetPos.add(target.getDeltaMovement().scale(time));
        }
        
        Vec3 direction = targetPos.subtract(muzzlePos).normalize();
        
        // 消耗能量
        base.consumeEnergy(energyCost);
        
        // 消耗弹药（支持弹药回收插件）
        if (base.hasAmmoRecyclingPlugin()) {
            if (base.getLevel().random.nextFloat() >= Config.ammoRecycleChance) {
                consumeAmmo(base);
            }
        } else {
            consumeAmmo(base);
        }
        
        // 创建子弹实体
        XxxEntity bullet = new XxxEntity(level, muzzlePos.x, muzzlePos.y, muzzlePos.z, damage);
        bullet.setOwner(null);
        bullet.setSourcePos(pos);
        bullet.setBasePos(pos.relative(facing.getOpposite()));
        bullet.shoot(direction, (float) BULLET_SPEED);
        level.addFreshEntity(bullet);
    }
    
    // ========== GeckoLib实现 ==========
    @Override
    public void registerControllers(AnimatableManager.ControllerRegistrar controllers) {
        controllers.add(new AnimationController<>(this, "controller", 0, state -> PlayState.CONTINUE));
    }
    
    @Override
    public AnimatableInstanceCache getAnimatableInstanceCache() {
        return this.cache;
    }
    
    // ========== NBT存储 ==========
    @Override
    protected void saveAdditional(@NotNull CompoundTag tag) {
        super.saveAdditional(tag);
        tag.putInt("Cooldown", cooldown);
        tag.putFloat("YRot0", yRot0);
        tag.putFloat("XRot0", xRot0);
    }
    
    @Override
    public void load(@NotNull CompoundTag tag) {
        super.load(tag);
        if (tag.contains("Cooldown")) cooldown = tag.getInt("Cooldown");
        if (tag.contains("YRot0")) yRot0 = tag.getFloat("YRot0");
        if (tag.contains("XRot0")) xRot0 = tag.getFloat("XRot0");
    }
}
```

public boolean hasTurretOnFace(Direction face) {
    Level level = getLevel();
    if (level == null) return false;
    BlockPos turretPos = getBlockPos().relative(face);
    BlockEntity be = level.getBlockEntity(turretPos);
    
    // 现有炮塔检测...
    if (be instanceof MachineGunTurretBlockEntity turret) { ... }
    if (be instanceof RailgunTurretBlockEntity turret) { ... }
    
    // 新炮塔检测（必须添加！）
    if (be instanceof XxxTurretBlockEntity turret) {
        TurretBaseBlockEntity base = turret.getBaseEntity();
        return base != null && base.getBlockPos().equals(getBlockPos());
    }
    return false;
}

public class XxxTurretGeoRenderer extends GeoBlockRenderer<XxxTurretBlockEntity> {
    
    public XxxTurretGeoRenderer(BlockEntityRendererProvider.Context context) {
        super(new XxxTurretGeoModel());
    }
    
    @Override
    protected void rotateBlock(Direction facing, PoseStack poseStack) {
        switch (facing) {
            case NORTH -> {
                poseStack.translate(0, 0.5, 0.5);
                poseStack.mulPose(Axis.XP.rotationDegrees(270));
            }
            case EAST -> {
                poseStack.translate(-0.5, 0.5, 0);
                poseStack.mulPose(Axis.ZP.rotationDegrees(-90));
            }
            case SOUTH -> {
                poseStack.translate(0, 0.5, -0.5);
                poseStack.mulPose(Axis.XP.rotationDegrees(90));
            }
            case WEST -> {
                poseStack.translate(0.5, 0.5, 0);
                poseStack.mulPose(Axis.ZP.rotationDegrees(90));
            }
            case DOWN -> {
                poseStack.translate(0, 1, 0);
                poseStack.mulPose(Axis.XP.rotationDegrees(180));
            }
            case UP -> { }
        }
    }
}

## ⚠️ 已踩过的坑

### 坑 1: GeckoLib数据票未注册导致崩溃

**现象**: 放置炮塔时游戏崩溃
**错误**: `NullPointerException: SerializableDataTicket.id() is null`
**原因**: 炮塔BlockEntity中声明了数据票但未在 `TurretMod.registerDataTickets()` 中注册
**解决**: 在 `TurretMod.java` 的 `registerDataTickets()` 方法中注册所有数据票

### 坑 2: 基座检测不到炮塔导致不攻击

**现象**: 炮塔能瞄准但不攻击，`isFaceEnabled()` 始终返回 false
**原因**: `TurretBaseBlockEntity.hasTurretOnFace()` 方法没有检查新炮塔类型
**解决**: 在 `hasTurretOnFace()` 方法中添加新炮塔的 `instanceof` 检查

### 坑 3: 缺少弹药检查导致不攻击

**现象**: 炮塔有能量、有目标、能瞄准，但不发射子弹
**原因**: `canShoot()` 方法缺少弹药检查，`shoot()` 方法缺少弹药消耗
**解决**: 
1. 添加 `hasAmmo()` 方法检查弹药槽
2. 添加 `consumeAmmo()` 方法消耗弹药
3. 在 `canShoot()` 中调用 `hasAmmo()`
4. 在 `shoot()` 中调用 `consumeAmmo()`

### 坑 4: 渲染器和GeoModel未创建导致模型不显示

**现象**: 放置炮塔后模型不显示
**原因**: 缺少 `XxxTurretGeoModel.java` 和 `XxxTurretGeoRenderer.java`
**解决**: 创建这两个类并在 `ClientEvents` 中注册

### 坑 5: 创造模式物品栏找不到物品

**现象**: 物品在创造模式物品栏找不到
**原因**: `ModCreativeModeTabs.java` 中未调用 `output.accept()` 添加物品
**解决**: 在创造模式标签页注册中添加所有新物品

### 坑 6: BlockEntity缺少旋转角度字段

**现象**: GeoModel编译错误 `yRot0 cannot be resolved`
**原因**: BlockEntity缺少 `yRot0` 和 `xRot0` 字段
**解决**: 在BlockEntity中添加这两个字段并实现NBT存储
