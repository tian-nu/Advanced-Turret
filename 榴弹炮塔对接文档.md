# 榴弹炮塔对接文档

## 一、当前问题

### 1.1 索敌问题
**现象**：炮塔只是单纯的附近有什么打什么，即使目标路径有方块阻挡仍然会一直攻击。

**根本原因**：
- `canReachWithParabola()` 方法虽然实现了抛物线路径模拟，但存在问题
- 模拟检测的逻辑不正确：在模拟过程中发现方块碰撞后，检查条件 `currentPos.distanceTo(start) < start.distanceTo(end) * 0.8` 不合理
- 该条件允许在接近目标时碰撞方块仍然返回 true，但这不应该发生

**相关代码位置**：
- 文件：`src/main/java/com/tian_nu/AdvancedTurret/blocks/entitys/GrenadeLauncherTurretBlockEntity.java`
- 方法：`canReachWithParabola()` (第389-437行)
- 调用位置：`isValidTarget()` 第588行

### 1.2 炮塔转向问题
**现象**：炮塔口不会对准发射路径的方向。

**根本原因**：
- 动画系统 `GrenadeLauncherTurretGeoModel.java` 计算角度时使用的是**直线方向**（目标位置 - 炮塔中心）
- 而发射方向应该是**抛物线初速度方向**
- 当前实现没有使用 `getAimDirection()` 方法返回的方向

**相关代码位置**：
- 文件：`src/main/java/com/tian_nu/AdvancedTurret/client/models/GrenadeLauncherTurretGeoModel.java`
- 方法：`setCustomAnimations()` (第37-78行)
- 问题代码：
```java
// 当前使用直线方向
Vector3f deltaPos = getTransform(direction).transform(
    new Vec3(targetPos.x - center.x, targetPos.y - center.y, targetPos.z - center.z).toVector3f()
);
```

## 三、修复方案

### 3.1 修复索敌问题

**方案**：重写 `canReachWithParabola()` 方法

```java
private boolean canReachWithParabola(LivingEntity target, Level level, BlockPos pos) {
    BlockState state = getBlockState();
    if (!(state.getBlock() instanceof GrenadeLauncherTurretBlock)) return false;
    Direction facing = state.getValue(GrenadeLauncherTurretBlock.FACING);
    
    Vec3 start = calculateMuzzlePosition(pos, facing);
    Vec3 end = target.position();
    
    Vec3 diff = end.subtract(start);
    double horizontalDist = Math.sqrt(diff.x * diff.x + diff.z * diff.z);
    
    // 超出射程
    if (horizontalDist > MAX_EFFECTIVE_RANGE) return false;
    
    // 计算抛物线初速度
    Vec3 velocity = calculateParabolicVelocity(start, end, BULLET_SPEED);
    
    // 模拟抛物线路径
    Vec3 currentPos = start;
    
    for (int tick = 0; tick < 100; tick++) {
        // 更新位置
        currentPos = currentPos.add(velocity);
        velocity = new Vec3(velocity.x, velocity.y - GRAVITY, velocity.z);
        
        // 检查是否到达目标附近（允许小误差）
        if (currentPos.distanceTo(end) < 1.5) {
            return true;
        }
        
        // 检查是否碰撞方块
        BlockPos checkPos = new BlockPos(
            (int)Math.floor(currentPos.x), 
            (int)Math.floor(currentPos.y), 
            (int)Math.floor(currentPos.z)
        );
        
        if (level.isLoaded(checkPos)) {
            BlockState blockState = level.getBlockState(checkPos);
            if (blockState.isSolidRender(level, checkPos)) {
                // 碰撞到方块，无法到达目标
                return false;
            }
        }
        
        // 检查是否飞出世界
        if (currentPos.y < level.getMinBuildHeight() || currentPos.y > level.getMaxBuildHeight()) {
            return false;
        }
    }
    
    return false;
}
```

## 六、物理公式参考

### 抛物线计算
```
已知：
- 起点 S = (x1, y1, z1)
- 终点 E = (x2, y2, z2)
- 重力 g = 0.05
- 抛物线高度 H（相对于起点）

求初速度 (vx, vy, vz)：

水平距离 d = sqrt((x2-x1)² + (z2-z1)²)
高度差 h = y2 - y1

上升时间 t_rise = sqrt(2 * H / g)
下降高度 = H - h
下降时间 t_fall = sqrt(2 * (H-h) / g)
总时间 t = t_rise + t_fall

水平速度：
vx = (x2 - x1) / t
vz = (z2 - z1) / t

垂直初速度：
vy = g * t_rise
```

### 角度计算
```
已知方向向量 (dx, dy, dz)

水平距离 h_dist = sqrt(dx² + dz²)

Y轴旋转（偏航）：
yRot = -atan2(dx, dz)

X轴旋转（俯仰）：
xRot = -atan2(dy, h_dist)
```
