## 1. 基座系统

### 1.1 基座等级

| 等级 | 能量容量 | 升级槽/面 | 功能槽 | 特性 |
|------|----------|-----------|--------|------|
| **T1** | 10,000 FE | 0 | 0 | 基础型号，仅支持弹药 |
| **T2** | 40,000 FE | 1 | 1 | 解锁升级槽和功能槽 |
| **T3** | 100,000 FE | 2 | 1 | 双升级槽 |
| **T4** | 250,000 FE | 2 | **2** | 双功能槽 |
| **T5** | 500,000 FE | 3 | **2** | 顶级配置 |

### 1.2 槽位结构

- **弹药槽**: 9格（全等级统一）
- **功能插件槽**: 
  - T1: 无
  - T2-T3: 1格
  - T4-T5: **2格**
- **升级组件槽**: 按面独立管理（6面各一套）
  - 每槽最多堆叠4个同类型组件

### 1.3 基座交互

- 右键打开主GUI
- 炮塔放置在基座的任意一面（6个方向）
- 支持多个炮塔同时运作（每面一个）
- 能量可通过Forge能量系统输入（FE）

## 2. 炮塔类型

### 2.1 炮塔参数表

| 炮塔类型 | 基础伤害 | 能耗/发 | 射速 | 攻击范围 | 特殊效果 |
|----------|----------|---------|------|----------|----------|
| **机枪** | 3.0 | 100 FE | 4发/s | 32格 | 无 |
| **磁轨炮** | 60.0 | 25,000 FE | 0.2发/s | 64格 | 穿透3个目标 |
| **激光** | 2.0/t | 1,000 FE/t | 20次/s | 32格 | 点燃目标 |
| **火箭** | 10+10* | 5,000 FE | 0.2发/s | 48格 | 爆炸半径4格 |
| **导弹** | 10+20* | 10,000 FE | 0.1发/s | 64格 | 追踪目标 |
| **榴弹** | 5+10* | 3,000 FE | 0.5发/s | 32格 | 抛物线弹道 |
| **垃圾** | 2.0 | 20 FE | 0.5发/s | 16格 | 任意物品作弹药 |
| **相位立场** | - | 100 FE/t | 持续 | 32格 | 敌方虚弱1+缓慢1 |
| **谐振立场** | - | 100 FE/t | 持续 | 64格 | 己方力量1+速度1 |

> `*` 表示范围伤害，在爆炸半径内生效

### 2.2 炮塔特殊机制

#### 机枪炮塔
- 基础入门级炮塔
- 高射速低伤害
- 使用通用子弹

#### 磁轨炮
- 高伤害单发武器
- **穿透机制**: 子弹可穿透最多3个目标
- 充能后发射

#### 激光炮塔
- **持续伤害**: 每tick造成伤害
- **点燃效果**: 目标燃烧3秒
- 光束视觉特效

#### 火箭炮塔
- 直线飞行火箭弹
- 爆炸伤害计算:
  - 直击伤害: 10
  - 爆炸伤害: 10（半径4格内）
  - 可破坏方块（需破坏插件）

#### 导弹炮塔
- **追踪机制**: 子弹自动追踪目标
- 高伤害+范围伤害
- 发射时有烟雾效果

#### 榴弹炮塔
- **抛物线弹道**: 重力影响
- 爆炸伤害计算:
  - 直击伤害: 5
  - 爆炸伤害: 10（半径3格内）

#### 垃圾炮塔
- **特殊弹药**: 接受任意物品作为弹药
- 低伤害高经济性
- 适合早期防御

#### 相位立场
- **立场效果**: 在范围内施加debuff
- 影响敌对生物
- 效果: 虚弱I + 缓慢I
- 持续消耗能量

#### 谐振立场
- **立场效果**: 在范围内施加buff
- 影响己方生物
- 效果: 力量I + 速度I
- 持续消耗能量

## 3. 弹药与子弹系统

### 3.1 子弹类型

| 子弹类型 | 对应炮塔 | 特性 |
|----------|----------|------|
| `TurretBulletEntity` | 机枪 | 直线飞行 |
| `RailgunBulletEntity` | 磁轨炮 | 穿透、高速 |
| `LaserBeamEntity` | 激光 | 光束、点燃 |
| `RocketEntity` | 火箭 | 爆炸 |
| `MissileEntity` | 导弹 | 追踪、爆炸 |
| `GrenadeEntity` | 榴弹 | 抛物线、爆炸 |
| `JunkProjectileEntity` | 垃圾 | 随机材质 |

### 3.2 无敌帧处理机制

**核心逻辑**（已实现）:
```java
// 子弹击中目标时
livingEntity.invulnerableTime = 0;  // 清除无敌帧
livingEntity.hurtTime = 0;          // 清除受伤动画时间
livingEntity.hurt(damageSource, damage);  // 造成伤害
```

**效果**: 炮塔子弹命中时绕过Minecraft默认无敌帧，防止多炮塔攻击时伤害丢失。

### 3.3 弹药匹配

- 机枪: 任意箭矢或子弹
- 磁轨炮: 专用磁轨
- 激光: 无需弹药（能量驱动）
- 火箭/导弹/榴弹: 专用弹药物品
- 垃圾: 任意物品

## 4. 升级组件系统

### 4.1 组件类型

| 组件 | 效果 | 叠加规则 |
|------|------|----------|
| **攻击增强组件** | 每层+10%伤害 | 最高300% |
| **射程扩展组件** | 每层+1格射程 | 最高32格 |
| **射速超频组件** | 每层-5%冷却 | 最低20% |
| **能效优化组件** | 每层-5%能耗 | 最低20% |
| **精度提升组件** | 减少散射 | 待实现 |

### 4.2 组件计算公式

```java
// 伤害计算
double multiplier = Math.min(3.0, 1.0 + (count * 0.10));

// 射程计算  
double range = Math.min(32.0, baseRange + count);

// 射速计算
double factor = Math.max(0.20, 1.0 - (count * 0.05));
int fireRate = Math.max(1, (int)(baseFireRate * factor));

// 能耗计算
double factor = Math.max(0.20, 1.0 - (count * 0.05));
int cost = Math.max(1, (int)Math.ceil(baseCost * factor));
```

### 4.3 槽位管理

- 按面独立管理（6面各有独立槽位）
- 每槽最多堆叠4个
- 只有该面有炮塔时槽位可见

## 5. 插件系统

### 5.1 功能插件列表

| 插件 | 效果 | 备注 |
|------|------|------|
| **智能芯片** | 目标识别配置 | 目标模式/黑白名单/友伤保护/预判瞄准 |
| **创造能量组件** | 无限能量 | 创造模式专用 |
| **太阳能插件** | 白天自动发电 | 速率待定 |
| **弹药回收插件** | 击杀回收弹药 | 概率待定 |
| **红石转化插件** | 红石信号转能量 | 待实现 |
| **破坏插件** | 允许炮塔破坏方块 | 新增 |

### 5.2 智能芯片配置

#### 目标模式
- `HOSTILE` - 仅敌对生物
- `NEUTRAL` - 仅中立生物
- `FRIENDLY` - 仅友善生物
- `PLAYERS` - 仅玩家
- `BLACKLIST_ONLY` - 仅黑名单目标
- `ALL` - 所有生物

#### 功能开关
- **友伤保护**: 不攻击主人及驯服生物
- **预判瞄准**: 根据目标移动计算提前量

#### 黑白名单
- 使用实体ID（如 `minecraft:zombie`）
- 黑名单优先级最高（强制攻击）
- 白名单优先级次高（强制不攻击）

### 5.3 破坏插件

**功能**: 允许炮塔攻击破坏方块

**行为**:
- 爆炸类炮塔（火箭/导弹/榴弹）可破坏方块
- 破坏强度取决于爆炸威力
- 被破坏方块掉落物品

**配置**:
- 可配置破坏强度
- 可配置是否保留掉落物

## 6. 智能目标识别

### 6.1 目标分类

| 分类 | 包含类型 | 检测方式 |
|------|----------|----------|
| **敌对** | 僵尸、骷髅、爬行者等 | `instanceof Enemy` |
| **中立** | 末影人、猪灵、狼等 | `instanceof NeutralMob` |
| **友善** | 动物、蝙蝠、鱼类 | `Animal/AmbientCreature/WaterAnimal` |
| **玩家** | 玩家 | `instanceof Player` |

### 6.2 目标选择逻辑

```
1. 检查是否在黑名单 → 强制攻击
2. 检查是否在白名单 → 强制跳过
3. 检查目标模式匹配
4. 检查友伤保护（主人/驯服生物）
5. 检查范围和视线
6. 选择最近目标
```

### 6.3 视线检测

- 检测三个点：头部、中心、脚部
- 任意一点可见即允许攻击
- 忽略炮塔自身方块碰撞

### 6.4 预判瞄准

```java
// 计算飞行时间
double time = distance / bulletSpeed;

// 预测目标位置
Vec3 predictedPos = targetPos.add(target.getDeltaMovement().scale(time));

// 调整射击方向
Vec3 aimDirection = predictedPos.subtract(muzzlePos).normalize();
```

## 9. 动画与视觉反馈

### 9.1 炮塔动画（GeckoLib）

- **瞄准动画**: 炮塔头部追踪目标
- **射击动画**: 开火时后坐力效果
- **待机动画**: 无目标时中立姿态

### 9.2 无电量动画提示

**新功能**: 炮塔在基座无电量时"低头"提示

```java
// 在炮塔BlockEntity中
if (base.getEnergyStored() == 0) {
    // 设置瞄准角度为向下
    setAnimData(TARGET_POS_Y, worldPosition.getY() - 1);
    setAnimData(TARGET_POS_X, worldPosition.getX() + 0.5);
    setAnimData(TARGET_POS_Z, worldPosition.getZ() + 0.5);
}
```

**视觉效果**: 炮塔头部朝下，模拟"休眠"状态

### 9.3 粒子效果

- 射击时: 烟雾粒子
- 激光命中: 火焰粒子
- 爆炸: 爆炸粒子
- 导弹追踪: 烟雾尾迹
