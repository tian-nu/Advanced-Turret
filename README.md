# 高级炮塔 (Advanced Turret)

一个 Minecraft 1.20.1 Forge 模组，基于开放式炮台设计理念，提供高度模块化的多炮塔防御系统。

## 主要特性

- **模块化炮塔系统**：多种炮塔类型（机枪、激光、导弹、火箭、磁轨、榴弹、共振场、相位场等），每种都有独特的攻击方式和视觉效果
- **多面升级**：炮塔底座支持 6 个面独立安装炮塔，可同时攻击不同方向的目标
- **智能芯片插件**：通过安装芯片启用高级功能，包括弹药循环、红石能量转换、太阳能发电、精准破坏等
- **伤害预扣协调**：多炮塔协同攻击时智能分配伤害，避免过量输出浪费能量
- **太阳能发电系统**：为炮塔提供可持续的清洁能源
- **红石能量转换**：将红石信号转换为炮塔能量
- **远程终端**：远程管理所有炮塔底座，支持批量配置
- **生物分析仪**：分析目标生物属性，辅助精准打击策略制定
- **自定义UI界面**：每面炮塔独立配置目标模式、黑名单/白名单、射程等参数

## 依赖

- **Minecraft**: 1.20.1
- **Forge**: 47.4.10+
- **GeckoLib**: 4.4+
